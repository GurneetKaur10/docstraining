Development Workflow Steps :-
1. Create a folder named hello_docker
2. Open in VS code.
3. Create file app.js
 { write the code : console.log("Hi, Welcome to Docker");
terminal : node app.js
it will print : Hi, Welcome to Docker}
4. Create new file in VS code named as Dockerfile and it will give a tip for install - install that.
Dockerfile Content :
FROM node:alpine
COPY . /app
WORKDIR /app
CMD node app.js
save this and check the version of Docker.
5. Use command "docker build -t hello_docker" . Press enter. (build will be created.)
6. Use command "docker image ls". (this will show image formed.)
7. Use command "docker run hello_docker". (to checker whether the docker is working or not)
